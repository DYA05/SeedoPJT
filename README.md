# Seedo

## 🔍 백엔드 초기 환경세팅

### • 라이브러리 설치

### 1. 가상환경 생성(anaconda) 및 확인

- 환경이름: seedo

```
conda create -n seedo python=3.11
conda env list
```

### 2. 가상환경 이동

```
conda activate seedo
```

### 3. pip install upgrade

```
pip install --upgrade pip
```

### 4. requirements.txt 설치

```
pip install -r requirements.txt
```
